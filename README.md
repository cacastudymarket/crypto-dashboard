# 🪙 Crypto Dashboard

A responsive crypto market dashboard built with vanilla HTML, CSS, and JavaScript.
Live data powered by CoinGecko API.

🔗 **Live Demo:** https://cacastudymarket.github.io/crypto-dashboard/

## Features

- 📊 Live prices for top 20 cryptocurrencies
- 😨 Fear & Greed Index (market sentiment indicator)
- 🔍 Search & filter coins
- 💱 USD / IDR currency toggle
- 📈 7-day price chart per coin (click any card)
- ⭐ Personal watchlist (saved to localStorage)

## Tech Stack

- HTML5, CSS3, JavaScript (ES6+)
- CoinGecko API (free tier)
- Alternative.me API (Fear & Greed)
- Chart.js (price visualization)

## How to Run

1. Clone this repo
2. Open `index.html` with Live Server
3. Done!
